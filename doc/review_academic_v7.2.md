- **推奨**:

#### 🔴 **Goal Termの定式化の矛盾（重要）**

**問題**: VAEが状態を予測しないのに、Goal Termでq(s_{t+1}|u)を使用

**箇所**:
- Line 118: `F(u) = D_{KL}[q(s_{t+1}|u) || p(s)]`
- Line 489: `Φ_goal(u) = D_{KL}[q(s_{t+1}|u) || p(s)]`
- Line 502: "where μ_{x,t+1}(u)はVAEが予測する次時刻の位置"

**矛盾点**:
- VAEはSPMのみを予測（Line 129, 879）
- 次状態は動力学モデル（RK4）で計算（Algorithm 1, Line 10）
- しかし、Goal Termではq(s_{t+1}|u)を使用

**推奨修正**:
```math
# 修正案1: 動力学モデルから計算
s_{t+1} = dynamics_rk4(s_t, u, dt, params)
Φ_goal(u) = D_{KL}[δ(s_{t+1} - dynamics_rk4(...)) || p(s)]
         ≈ (P_pred - P_target)^2 / (2σ_P^2)
where P_pred = v_{t+1} · d_goal, v_{t+1} from dynamics_rk4

# 修正案2: 記号を変更
q(s_{t+1}|u) → p(s_{t+1}|s_t, u)  # 動力学モデルによる確定的予測
```

#### ⚠️ **理論的整合性の問題**

**問題1: Active Inference原理との整合性**
